# Myblog

#### 介绍
​		Myblog是一个前后端分离的个人博客,可用于个人博客建站使用和SpringBoot+Vue开发练习.

Myblog在线体验[HLBolog](huanglin.online)

#### 技术栈/版本介绍:

* 博客后端
  * JDK 1.8
  * SpringBoot 2.2.6
  * Mybatis 1.3.2
  * Mysql 5.7
  * Druid 1.1.17
  * Shiro 1.4
  * Jwt 3.2.0
  * Commonmark 0.10.0
  * Bytebuddy 1.10.9
  * Fastdfs 1.26.1
  * Swagger 2.9.2
  * Knife4j 2.0.4
  * Jsoup 1.10.2
  * Log4j 1.2.17 
* 博客前端
  * Webpack 3.6.0
  * Vue 2.5.2
  * Semantic UI - 2.4.2
